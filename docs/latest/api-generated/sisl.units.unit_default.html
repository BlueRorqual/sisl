unit_default¶

sisl.units.unit_default(group, tbl=None)[source]¶

The default unit of the unit group group.

Parameters:

group : str

look-up in the table for the default unit.

tbl : dict, optional

dictionary of units (default to the global table)

Examples

>>> unit_default('energy')
'eV'
